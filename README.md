# Habit-Analytics-Dashboard

A Google Sheets based dashboard for tracking daily habits and analyzing performance.
## Live Dashboard

[View Habit Analytics Dashboard](https://docs.google.com/spreadsheets/d/1zaSbxJ5qQ8HpH075xpa0ZD15d4sr6JMd88FfGl7GyOo/edit?usp=sharing)
## Features

- Daily habit tracking
- Weekly progress tracking
- Monthly analytics dashboard
- Habit ranking system
- Streak tracking
- Completion percentage calculation
- Goal progress monitoring

## Tools Used

- Google Sheets
- Formulas
- Conditional Formatting
- Charts & Dashboards
